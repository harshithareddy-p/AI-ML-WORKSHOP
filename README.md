# House Cost Clustering and Visualization

This project demonstrates K-Means clustering using Python, Pandas, Matplotlib, and Scikit-learn.

## Features
- Elbow Method for optimal clusters
- K-Means clustering
- Data visualization
- Random graph visualization

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- NumPy

## Output
Includes clustering plots and sample visualizations.
